# Customer Churn & Retention Analysis

## Overview

This project analyzes customer churn and retention patterns to identify why customers leave, what drives long-term engagement, and how businesses can improve customer retention. An interactive dashboard was created to support data-driven decision-making.

## Objectives

* Measure churn and retention rates
* Identify high-risk customer segments
* Analyze customer lifetime trends
* Discover key retention drivers
* Provide actionable business recommendations

## Tools Used

* Power BI
* Power Query
* DAX
* Excel

## Key Metrics

* Total Customers
* Churn Rate
* Retention Rate
* Average Customer Tenure
* Customer Lifetime Trends

## Key Insights

* Customers with shorter tenure are more likely to churn.
* Long-term contracts show higher retention rates.
* Certain customer segments contribute disproportionately to churn.
* Improved engagement can significantly increase customer retention.

## Recommendations

* Strengthen customer onboarding.
* Promote long-term subscription plans.
* Target high-risk customers with retention campaigns.
* Monitor churn trends regularly using dashboards.

## Skills Demonstrated

Customer Analytics • Churn Analysis • Cohort Analysis • KPI Development • Data Visualization • Business Intelligence
